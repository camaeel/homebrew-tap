# Camaeel Tap

## How do I install these formulae?

`brew install camaeel/tap/<formula>`

Or `brew tap camaeel/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "camaeel/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
